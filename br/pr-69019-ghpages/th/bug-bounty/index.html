โปรแกรมรางวัลบั๊ก

ความปลอดภัยคือความร่วมมือ รายงานจุดบกพร่องและรับรางวัล

เกี่ยวกับ Deriv

เราให้บริการซื้อขายออนไลน์แก่ลูกค้ากว่า 2 ล้านรายผ่านเดสก์ท็อปและแอปพลิเคชันมือถือในหลายแพลตฟอร์ม

ความปลอดภัยเป็นสิ่งสำคัญสำหรับเรา เราปรับปรุงผลิตภัณฑ์และบริการของเราอย่างต่อเนื่องโดยร่วมมือกับนักวิจัยด้านความปลอดภัยอิสระทั่วโลก

ขอบเขต

โปรแกรมนี้ครอบคลุมเฉพาะสินทรัพย์ Deriv หากคุณพบข้อบกพร่องในแอปของบุคคลที่สาม โปรดรายงานให้เจ้าของที่เกี่ยวข้องทราบ คุณสามารถส่งรายงานข้อบกพร่องที่เกี่ยวข้องกับแง่มุมต่อไปนี้ของธุรกิจของเรา:

ธุรกิจสำคัญ

ไซต์การชำระเงินของเรา: cashier.deriv.com

ไซต์เข้าสู่ระบบของเรา: oauth.deriv.com

WebSockets API ของเรา: *.binaryws.com api.deriv.com

แพลตฟอร์มการซื้อขายหลักของเรา: app.deriv.com*

*สิ่งนี้ครอบคลุมเฉพาะฟังก์ชันที่จัดการโดย Deriv

แพลตฟอร์มการซื้อขายเดิมของเรา: smarttrader.deriv.com
